# Dodge

Simple game where you dodge projectiles, made with Lua and Löve2D.

## Installation

You need git and [Löve2D](https://love2d.org/) installed.

```sh
$ git clone https://github.com/vanillaiice/dodge-love2d
$ cd dodge
$ love .
# or
$ make
```

## Game Controls

`Up` - Move up

`Down` - Move down

`Right` and `Left` - Rotate

`Space` - Pause

`Backspace` - Restart

`Esc` - Quit
